 📝 Project Description

The **Task Automation with Python Scripts** project automates a simple real-life repetitive task using Python. This project can perform tasks such as moving image files, extracting email addresses from a text file, or scraping the title of a webpage. It helps improve productivity by reducing manual work.

 🎯 Features

* 📂 Automates repetitive file-related tasks.
* 🖼️ Moves all .jpg files from one folder to another.
* 📧 Extracts email addresses from a .txt file.
* 🌐 Scrapes the title of a fixed webpage.
* 💾 Saves the output to a new file.

 🛠️ Technologies Used

* 🐍 Python
* 📁 OS Module (os)
* 📦 Shutil Module (shutil)
* 🔍 Regular Expressions (re)
* 🌐 Requests Module (requests)
* 📂 File Handling
